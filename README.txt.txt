========================================
1. ЧТО ЭТО ЗА САЙТ
========================================

Это официальный сайт футбольного клуба Deportivo Moscu.

Сайт сделан в стиле современного футбольного клуба:
- белый фон
- зелёно-чёрная айдентика
- логотип Deportivo Moscu
- крупные заголовки
- карточки игроков
- новости
- таблица и матчи
- раздел клуба
- академия
- медиа
- магазин

Сайт поддерживает 3 языка:
- русский
- английский
- испанский

Переключение языков работает через кнопки RU / EN / ES в верхнем меню.

В этом файле можно менять:

- описание новости

4. ПАПКА IMAGES
========================================

Папка images должна находиться в главной папке сайта.

Внутри обязательно должен быть файл:

images/logo.png

Это логотип Deportivo Moscu.

Если логотип не показывается, проверить:
1. папка называется exactly: images
2. файл называется exactly: logo.png
3. путь в коде такой: images/logo.png

6. ЧТО ДЕЛАТЬ ЕСЛИ БЕЛЫЙ ЭКРАН
========================================

1. Проверить, что файл сохранён.
2. Нажать Ctrl + F5 в браузере.
3. Проверить, что файл лежит в главной папке сайта.
4. Проверить название файла:
   - index.html
   - news.html
   - team.html
   - league.html
   - club.html
   - academy.html
   - media.html
   - store.html
5. Проверить, что нужный JS-файл лежит рядом:
   - players-data.js
   - news-data.js
   - league-data.js
6. Проверить, что логотип лежит здесь:
   images/logo.png

7. ЧТО МОЖНО МЕНЯТЬ В БУДУЩЕМ
========================================

Игроков:
менять в players-data.js

Новости:
менять в news-data.js

Таблицу и матчи:
менять в league-data.js

Логотип:
заменить файл images/logo.png на новый файл с таким же названием

Товары магазина:
менять внутри store.html в массиве products

Тексты клуба:
менять внутри club.html

Тексты академии:
менять внутри academy.html

Медиа и галерею:
менять внутри media.html
